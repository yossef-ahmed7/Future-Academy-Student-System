# Future Academy Student System

## Overview
A C++ console application built using Object-Oriented Programming (OOP).

## Features
- Add new students
- Login using Student ID
- Display student information
- Show subjects based on:
  - Department
  - Level
  - Semester

## Technologies
- C++
- Object-Oriented Programming (OOP)
- Visual Studio

## Concepts Used
- Classes
- Objects
- Inheritance
- Encapsulation
- Arrays
- Functions
